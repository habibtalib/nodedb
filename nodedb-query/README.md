# `nodedb-query`

> Cross-engine query primitives: BM25, RRF fusion, text analysis

Shared scoring and analysis utilities used by both the FTS engine and hybrid query paths. Holds the BM25 implementation, language-aware analyzers, and reciprocal-rank fusion used by GraphRAG and hybrid vector + text queries.

## Status

Pre-1.0. APIs may change between minor versions until 1.0. See the
workspace [`README.md`](../README.md) for the full project overview and
the GitHub [release notes](https://github.com/NodeDB-Lab/nodedb/releases)
for per-version changes.
