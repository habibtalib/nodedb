# `nodedb-raft`

> Raft consensus: leader election, log replication, snapshots

Self-contained Raft implementation used by NodeDB clusters. Leader election, log replication, joint-consensus reconfiguration, InstallSnapshot with chunked transport. Designed for embedding inside the Control Plane.

## Status

Pre-1.0. APIs may change between minor versions until 1.0. See the
workspace [`README.md`](../README.md) for the full project overview and
the GitHub [release notes](https://github.com/NodeDB-Lab/nodedb/releases)
for per-version changes.
