# `nodedb-graph`

> CSR adjacency index, 13 graph algorithms, MATCH pattern engine

Graph traversal and analytics shared between Origin and Lite. PageRank, WCC, SSSP, Louvain, k-Core, betweenness, etc., plus a Cypher-subset MATCH executor and OLAP graph snapshots for distributed BSP.

## Status

Pre-1.0. APIs may change between minor versions until 1.0. See the
workspace [`README.md`](../README.md) for the full project overview and
the GitHub [release notes](https://github.com/NodeDB-Lab/nodedb/releases)
for per-version changes.
